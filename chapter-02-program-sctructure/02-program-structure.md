# Chapter 2: Program Structure

**Overview**
This chapter introduces the basic building blocks of JavaScript programs: expressions, statements, and control flow. You will learn how to direct the flow of your program using `if`, `else`, `while`, and `for` loops.

## Key Concepts

- **Expressions vs. Statements**: Expressions produce values; statements perform actions.
- **Control Flow**: Using `if`/`else` for branching logic.
- **Loops**: Repeating code blocks with `while` and `for`.
- **Variables**: Declaring with `let` and `const`.

### 2. FizzBuzz

Write a program that uses `console.log` to print all the numbers from 1 to 100, with two exceptions. For numbers divisible by 3, print "Fizz" instead of the number. For numbers divisible by 5 (and not 3), print "Buzz" instead.

### 3. Chessboard

Write a program that creates a string that represents an 8×8 grid, using newline characters to separate lines. At each position of the grid, there is either a space or a "#" character. The characters should form a chessboard.

When you have a working program, make it accept a parameter `n` that specifies the width and height of the board.
